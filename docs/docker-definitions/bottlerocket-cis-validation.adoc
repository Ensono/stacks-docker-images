== Bottlerocket CIS validation

This image is used to validate an AWS EKS cluster to ensure the Bottlerocket OS complies with CIS level 2. 

=== Specs

**Platforms**: `linux/amd64`, `linux/arm64`

**Base Image**: ubuntu:22.04
